Tag Viewer

The Tag Viewer allows you to view tag information and raw SWF bytes of each tag in the file.

Tags

The tags are listed in order from first to last in the left hand column of the tab. Clicking on a tag will display the information in the Tag Information column.
If the tag has a red "E" as its icon, then it means there was an error attempting to parse that tag. See the tag for additional information.


Tag Information

For each tag, you will be able to see the basic tag properties along with the hex bytes for that tag. For some supported tags, the pane will display an XML summary of the tag in the same format as the Flex SDK swfdump tool.
