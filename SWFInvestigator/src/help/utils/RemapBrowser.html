Remap Browser

This functionality exposes sandbox remapping in a primitive format. This will allow you to remap a directory on your hard drive to a remote domain using the WebKit browser.


Content Directory

This is the directory on your hard drive that contains the files to be remapped.


Remote Domain

This is the remote domain that the content will be remapped to before it is loaded into the Webkit browser.


Index Page

This the first page that the WebKit browser should load from the directory.
