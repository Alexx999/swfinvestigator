SWF Info

This tab presents some basic overview of the SWF and information that can be obtained from its header and the File Attributes tag. It also displays the exported classes and file tags included from the SWF.

Header Section

The top section of this panel includes information from the SWF's header and the file attributes tag. Information regarding the File Attibutes tag and the meaning of the flags can be found within the SWF File Format Specification posted at the SWF Technology Center

Exported Classes

Contains the list of exported classes (if any) from the SWF.

File Tags

The File Tags section provides the integer value for the file tag, its name and the number of times it occurs within the SWF.
