Version 0.6.2
  • Added double-click functionality to the Strings tab to allow the user to copy the string to the clipboard.
  • Added cross-domain tester to allow testing of cross-domain policy files.
  • Added warning for LZMA files. LZMA support will be included in the next version.
  • Added support for comma-delimited hex to the encode/decode utility.
  • Bug fixes in the binary editor.
Version 0.6.1
  • Initial public beta.
  • Includes features to view SWF Tags, disassembly and binary data.
  • Includes the ability to dynamically interact with a SWF as it is running.
  • Includes security tools for identifying cross-site scripting vulnerabilities and testing AMF services.
  • Includes the ability to view data relating to the SWF in LSOs and settings files.
  • Includes ancillary tools such as a Mini Web Server and a simple AS3 Compiler.
  • Includes auto-update capabilities.
