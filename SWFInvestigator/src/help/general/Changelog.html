Version 0.6.5
  • Changed MethodInfo.as to show String index.
  • Changed Binary Editor (StringToBinFile.as) to accept negative numbers and fixed error display issue.
  • Altered TypeName to handle objects with no name (TypeName.as).
  • Changed TagValues, EnableTelemetry, StartSound2, TagHandler, TagDecoder, SwfxPrinter, SWF_Constants, SWF.as for the EnableTelemetry and StartSound2 tags.
  • Updated EncDecUtils.mxml to fix a bug where writeUTF was used instead of writeUTFBytes.
  • Fixed a bug in AMFTransmitter.mxml, AMFFuzzer.mxml so that it will send multiple parameters.
  • Increased the width of the URL dialogue in GetSWFUrl.as to increase readability.
Version 0.6.4
  • Fixed critical bug in the Hex Editor.
  • Fixed Mac install issue.
Version 0.6.3
  • Added LZMA support for the SWF disassembler
  • Added LZMA support to the binary editor.
  • Fixed bug in the ASDecompiler.
  • Changed the name of help Tabs directory to lower case.
Version 0.6.2
  • Added double-click functionality to the Strings tab to allow the user to copy the string to the clipboard.
  • Added cross-domain tester to allow testing of cross-domain policy files.
  • Added warning for LZMA files. LZMA support will be included in the next version.
  • Added support for comma-delimited hex to the encode/decode utility.
  • Bug fixes in the binary editor.
Version 0.6.1
  • Initial public beta.
  • Includes features to view SWF Tags, disassembly and binary data.
  • Includes the ability to dynamically interact with a SWF as it is running.
  • Includes security tools for identifying cross-site scripting vulnerabilities and testing AMF services.
  • Includes the ability to view data relating to the SWF in LSOs and settings files.
  • Includes ancillary tools such as a Mini Web Server and a simple AS3 Compiler.
  • Includes auto-update capabilities.
