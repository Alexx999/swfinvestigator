Entering Object and Class data into a paramter entry field

Entering Object Data

If you want to send an object to the SWF via a parameter entry field, it is not necessary to enter in any value. Just select Object from the pull down menu and a new panel will appear.


Within the new panel, you can enter in the name and value for each property of the object and select its data type. If you would like to delete a property, then just delete the value in the name column for the entry that you want to delete. Once de-selected, the row will be emptied. Click Create once the object is defined.

Entering Class Data

OE Send & OE Receive

If you want to send the created object to the Object Editor for saving, you would use the OE Send button. If you want to import an object from the Object Editor, you would use the OE Retrieve button. If you retrieve an object from the Object Editor, then the panel is immediately closed. Either you create an object or you retrieve an object.
